# 🛡️ Intruder Detection System (Image Processing)

## 📌 Overview
This project is an AI-based surveillance system that detects human presence in real-time using computer vision techniques. It identifies intruders and can trigger alerts.

---

## 🚀 Features
- Real-time human detection using YOLO
- Intruder identification system
- Image capture for detected persons
- Alert system integration (Telegram)
- Reduced false detection using confidence filtering

---

## 🧠 Technologies Used
- Python
- OpenCV
- YOLOv8
- NumPy

---

## ⚙️ Working
1. Capture live video from camera  
2. Detect humans using YOLO  
3. Check if person is authorized  
4. If unknown → mark as intruder  
5. Send alert with captured image
